== Description ==

Provides a pair of popup charts: the Yi index and the Yi radical index.

== What it does: ==


Pops up either of the charts when clicking on their links. Currently fits into a sidebar. Created to fit into the Multilingual-Dictionary theme's Search Bar Popups sidebar. (You can find the code in the theme's searchform.php and functions.php.)


== Installation ==

Installation Instructions:

1. Download the plugin and unzip it.
2. Put the yi plugin into your wp-content/plugins/ directory.
3. Go to the Plugins page in your WordPress Administration area, find the plugin, and click 'Activate'.
4. Go to Widgets. Drag into the sidebar of your choice.

== Known Issues ==

Currently, the background, font, and link colors match those of Multilingual-Dictionary. If you change themes, these colors will remain the same.
