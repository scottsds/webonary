=== Plugin Name ===
Special Characters Buttons
Contributors Philip Perry, SIL Global
Tags: bantu characters
Requires at least: 3.0
Tested up to:
Stable tag: 0.0

== Description ==

Provides buttons for special characters

== What it does: ==

When clicking on a button with a special character, it will insert it into the search box.


== Installation ==

Installation Instructions:

1. Download the plugin and unzip it.
2. Put the special characters plugin into your wp-content/plugins/ directory.
3. Go to the Plugins page in your WordPress Administration area, find the plugin, and click 'Activate'.
4. Go to Widgets. Drag into the sidebar Search Bar Popups
